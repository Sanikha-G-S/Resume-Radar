\# Resume Radar



Resume Radar is a web-based resume analysis platform built using Python and Streamlit.



\## Features



\- PDF Resume Upload

\- Resume Text Extraction

\- ATS Score Calculation

\- Skill Extraction

\- Job Description Matching

\- Missing Skill Detection

\- PDF Report Generation

\- Interactive Dashboard



\## Tech Stack



\- Python

\- Streamlit

\- PyMuPDF

\- Pandas

\- FPDF



\## Run Locally



```bash

streamlit run app/main.py
